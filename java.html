☕ Установка Java 17 и 21

Обычно лаунчер скачивает Java сам. Если возникли проблемы — установите обе версии ниже для полной совместимости.

Windows
Для систем Windows 10 и 11 (64-бит).
Ставим первой
Скачать Java 21
Ставим второй
Скачать Java 17
macOS
Выберите ваш тип процессора.
Apple Silicon (M1-M5) — ОБЕ

Intel (Старые Mac) — ОБЕ

Linux
Установка производится через терминал вашего дистрибутива.
Инструкция по установке
Важно: Мы настоятельно рекомендуем установить обе версии (17 и 21). Это гарантирует, что любые компоненты сервиса будут работать корректно без дополнительных настроек.
