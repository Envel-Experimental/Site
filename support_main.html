ИСПРАВЛЕНИЕ ПРОБЛЕМЫ НА MACOS на M ПРОЦЕССОРАХ APPLE (~2% пользователей)

Проблема

  1. Версия игры 1.16.5 не поддерживает ARM процессоры.

Вариант решения

  1. Этот вариант решения не тестировался.
  2. Сейчас используется Java ARM. Старые версии игры не поддерживают M процессоры. Нужно использовать Java x64 для того, чтобы сработала эмуляция rozetta. Следующий шаг предлагает конкретные действия для этого.
  3. Скачайте JAVA x64 MacOS: https://adoptium.net/temurin/releases/?os=mac&arch=x64&package=jre&version=8
    4. СКАЧАТЬ JAVA x64 MacOS
  4. Заходим в настройки лаунчера
  5. Java
  6. Выбираем исполняемый файл скачанной и установленной JAVA x64 MacOS. Он будет лежать по расположению java в bin\java
  7. Запускаем игру
  8. Информируйте о ЛЮБОМ результате. Нам важно знать, что это помогло/не помогло!
    9. СОЗДАТЬ ТИКЕТ GITHUB

ИСПРАВЛЕНИЕ ПРОБЛЕМЫ С ЗАГРУЗЧИКОМ FABRIC (ВСЕ СИСТЕМЫ) (<1% пользователей)

Проблема

  1. При входе вылетает окно Fabric Loader с любой ошибкой.

Вариант решения

  1. Установленные модификации для оптимизации не работают на текущем оборудовании. Проблема решается индивидуально и крайне редкая. Можно отключить все, это значительно снизит производительность, но решит проблему. Следующий шаг предлагает конкретные действия для этого.
  2. Заходим в настройки лаунчера
  3. Моды
  4. Отключаем все необязательные моды
  5. Запускаем игру
  6. Информируйте разработчика, если не помогло.
    7. СОЗДАТЬ ТИКЕТ GITHUB
