Установка на Linux 🐧

Простая инструкция для линукс.

Обязательно
Проверь Java

Обычно лаунчер сам скачивает Java. Если возникли проблемы — убедись, что в системе установлены JAVA 17 и JAVA 21.

Инструкция: Как установить/сменить Java

Шаг 1
Запуск файла

1. Скачай файл (.AppImage или нужный формат).

2. Сделай его исполняемым (Свойства -> Права -> Разрешить исполнение, или chmod +x).

3. Запусти файл.

Совет
Дистрибутивы

Если стандартный файл не работает, можно скачать установщик специально под твою версию Linux (DEB, RPM и др).
