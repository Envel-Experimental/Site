Установка Java 17 и 21

Обычно лаунчер скачивает всё сам, но если возникли проблемы — установите Java вручную:

Ubuntu / Debian / Mint

sudo apt install openjdk-17-jdk openjdk-21-jdk

Fedora / CentOS

sudo dnf install java-17-openjdk java-21-openjdk

Arch Linux / Manjaro

sudo pacman -S jdk17-openjdk jdk21-openjdk

Решая проблемы с безопасностью, ознакомьтесь с Отказом от ответственности.

Ошибки библиотек?

Если лаунчер не открывается, часто помогает полное обновление пакетов системы:

sudo apt update && sudo apt upgrade
sudo pacman -Syu
Системные требования

Убедитесь, что ваше оборудование соответствует требованиям.

🖥️
Видеокарта
  • NVIDIA: Серия 400+ (Советуем проприетарные драйверы)
  • AMD: Radeon HD 7000+ (Mesa)
  • Intel: Iris / HD (с 2019 года)
🧠
Память (RAM)
  • 8 ГБ — Рекомендуется
  • 4 ГБ — Минимум (возможны вылеты)
🐧
Система
  • Ядро (Kernel): 5.x и выше
  • Архитектура: Только x64
  • Поддержка OpenGL 4.4
Важно про NVIDIA

Если у вас карта NVIDIA, мы настоятельно рекомендуем использовать проприетарные драйверы (nvidia-driver).

На свободных драйверах (Nouveau) игра может работать с сильными лагами или артефактами.
